#3: Objects

Inheritance is also possible through prototypal inheritance:


  var Opticien = function(firstname, lastname) {
    Person.call(this, firstname, lastname); // super(firstname, lastname)
    this.job = "IT Consultant";
  };
  Opticien.prototype = new Person(); // Opticien extends from Person
  
  var me = new Opticien("Dimitri", "Mestdagh");
  me.sayJob();
  me.sayName();
  • Inheritance is ook mogelijk via prototypal inheritance
  • Dit doe je door als prototype een nieuwe instantie toe te kennen van het super-prototype
  • Bijvoorbeeld Opticien.prototype = new Person();
  • Je kan ook de constructor van het parent prototype uitvoeren, door Person.call() te gebruiken
