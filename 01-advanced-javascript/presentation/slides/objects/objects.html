#3: Objects

To create a new object, you can use:


  var me = {
    firstname: "Dimitri",
    name: "Mestdagh"
  };

But what about classes & constructors?

  • Dit is een vaak voorkomende syntax om een object te voorzien
  • Heel gelijkaardig aan JSON
  • Bij JavaScript is het niet vereist om de velden tussen quotes te plaatsen
  • Objecten zijn één ding, maar is het ook mogelijk om classes te definiëren in JavaScript
