#3: Objects

JavaScript uses prototypes:


  var Opticien = function(firstname, lastname) {
    this.firstname = firstname;
    this.lastname = lastname;
    
    this.sayName = function() {
      console.log(this.firstname + " " + this.lastname);
    };
  };
  
  var me = new Opticien("Dimitri", "Mestdagh");
  me.sayName();

Opticien appears to be a normal function, but it can be instantiated using the new keyword. We call this a prototype.

  • In plaats van classes, kennen we in JavaScript prototypes
  • Een prototype ziet eruit als een gewone functie
  • Maar kan geinstantieerd worden met het new keyword
  • Alles wat in de functie staat wordt uitgevoerd op het moment een nieuw object wordt aangemaakt
  • Dit is dus de constructor (= function constructor
  • Zowel de velden als de logica worden gedefinieerd tijdens het instantieren
