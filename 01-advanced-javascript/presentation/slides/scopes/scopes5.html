#2 Scopes

Each scope has access to its outer scope. There is also a "root" scope, which we call the global scope

  • In browsers this is window
  • On Node.js it is global

Be careful, if you don't use var in front of your variable, it will create the variable on the global scope!


  function foo() {
    a = "bar";
  }
  foo();
  console.log(window);
